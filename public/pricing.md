# Skarm pricing

Skarm is an AI-native issue tracker. Every workspace starts on the Free plan.
Paid plans lift the limits and unlock the AI agent. All limits are enforced
on the server, so they are authoritative rather than cosmetic.

## Plan comparison

| Capability | Free | Pro ($20/mo) | Enterprise ($99/mo) |
| --- | --- | --- | --- |
| Members | 3 | 10, then $10 per extra seat | Unlimited |
| Projects | 2 | Unlimited | Unlimited |
| Issues | 100 | Unlimited | Unlimited |
| Teams | Unlimited | Unlimited | Unlimited |
| Boards and cycles | Yes | Yes | Yes |
| Search (full text) | Yes | Yes | Yes |
| GitHub and Figma sync | Yes | Yes | Yes |
| Dependency graph | Yes | Yes | Yes |
| Email digests | Yes | Yes | Yes |
| AI agent | No | 50 messages per user per day | Unlimited |
| Priority support | No | No | Yes |

## Free

Best for trying Skarm or running a small team.

- 3 members
- 2 projects
- 100 issues
- All core tracking: issues, boards, cycles, search, templates, sharing
- GitHub and Figma integrations
- No AI agent

## Pro ($20 per month)

Best for growing product teams that want AI in the loop.

- 10 members included, then $10 per additional seat
- Unlimited projects
- Unlimited issues
- Everything in Free
- AI agent: chat, issue drafting, and triage, sharing a daily allowance of
  50 messages per user

## Enterprise ($99 per month)

Best for larger organizations that need scale and support.

- Unlimited members
- Unlimited projects
- Unlimited issues
- Everything in Pro
- Unlimited AI agent usage
- Priority support

## How limits work

- Limits apply per workspace (organization), not per user.
- Free-plan limits on seats, projects, and issues are enforced when you try
  to exceed them, with an in-app prompt to upgrade.
- The AI agent allowance on Pro is shared across chat and issue drafting and
  resets daily per user.
